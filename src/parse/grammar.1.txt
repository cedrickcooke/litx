S           -> Blocks EOF
            -> SWS Blocks EOF
Blocks     - > Block | Block SWS | Block SWS Blocks
Block       -> BlockItem | BlockItem Block
BlockItem   -> Text | AnyExpr
AnyExpr     -> Expr | Math | Comment
Text        -> TextItem | TextItem Text
TextItem    -> WS
            -> TT::Char | TT::Escaped | TT::KeyStart
            -> TT::Number | TT::Quote | TT::Word
Comment     -> TT::OpenComment TT::CloseComment
            -> TT::OpenComment CommentBody TT::CloseComment
CommentBody -> Comment | Comment CommentBody
            -> CommentTerm | CommentTerm CommentBody
CommentTerm -> TT::BlankLine | TT::Char
            -> TT::CloseExpression | TT::CloseMath
            -> TT::Escaped | TT::KeyStart
            -> TT::NewLine | TT::Number
            -> TT::OpenExpression | TT::OpenMath
            -> TT::Quote | TT::Space | TT::Word
Expr        -> TT::OpenExpr TT::CloseExpr
            -> TT::OpenExpr ExprBody TT::CloseExpr
ExprBody    -> AWS | AWS ExprBody
            -> ExprTerm | ExprTerm ExprBody
ExprTerm    -> ExprItem | ExprProp
ExprItem    -> ExpIdent | ExprLit | AnyExpr
ExprProp    -> TT::KeyStart ExprIdent AWS ExprItem
ExprIdent   -> TT::Word
ExprLiteral -> TT::Number | String
Math        -> TT::OpenMath TT::CloseMath
            -> TT::OpenMath MathBody TT::CloseMath
MathBody    -> AWS | AWS MathBody
            -> MathTerm | MathTerm MathBody
MathTerm    -> AnyExpr
            -> TT::Char | TT::Escaped | TT::KeyStart
            -> TT::Number | TT::Word
String      -> TT::Quote TT::Quote
            -> TT::Quote StringBody TT::Quote
StringBody  -> StringTerm | StringTerm StringBody
StringTerm  -> AWS
            -> TT::Word | TT::Char | TT::Number
            -> TT::Escaped | TT::KeyStart
            -> TT::CloseComment | TT::CloseExpression | TT::CloseMath
            -> TT::OpenComment | TT::OpenExpression | TT::OpenMath
AWS         -> WS | WS AWS
            -> SWS | SWS AWS
SWS         -> TT::BlankLine
WS          -> TT::Space | TT::Space WS
            -> TT::NewLine | TT::NewLine WS
